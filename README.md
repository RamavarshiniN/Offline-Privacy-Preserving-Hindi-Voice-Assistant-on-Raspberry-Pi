# Offline-Privacy-Preserving-Hindi-Voice-Assistant-on-Raspberry-Pi
Offline Hindi voice assistant on Raspberry Pi using Edge AI. Runs speech recognition, intent classification, and command execution fully on-device with Vosk and FastText. Ensures privacy, low latency, and no internet dependency while supporting device control, calculations, and date/time queries.

## Hardware Used

- Python  
- Vosk  
- FastText  
- Raspberry Pi OS  
- Edge AI  
- Embedded Linux

## Objectives

- Develop an offline Hindi speech recognition system
- Implement machine learning based intent classification
- Enable device control and information retrieval
- Ensure privacy-preserving on-device processing
- Optimize models for embedded deployment



## Methodology

1. Wake word detection activates the assistant
2. Speech input is captured and converted to text
3. FastText predicts the user intent
4. Fuzzy matching improves recognition accuracy
5. Commands are executed and results are displayed
